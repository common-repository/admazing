Add a fancy toolbar in your dashboard and customize your login screen

== Description ==

Add a **custom icons shortcuts** into your dashboard. Customize your login page with **live preview**.

= Features =

* **NEW**: Deactivate autosave posts & revisions + Add excerpt field to pages.
* Add a fancy toolbar on Dashboard: over the widgets or as another widget.
* It includes the **Font Awesome** (479 icons) and **Genericons Font** (130 icons) sets.
* **Configure & manage** the Dashboard Toolbar icons: action, size, color and order.
* **Customize the login** page (optional) with your logo, URL logo, backgrounds, input fields, button, etc. Solid colors and gradients on backgrounds.
* **Live preview** in login page customisation.
* **Clear front-end & performance**: This plugin don't load anything in your website, only in login and admin side.

= Font icons included =

WordPress Admin icons (since WP 3.8).
Font Awesome: [http://fortawesome.github.io/Font-Awesome/](http://fortawesome.github.io/Font-Awesome/).

== Installation ==

1. **Add the plug-in using wordpress admin, find or upload it, via website or FTP.**

2. **Activate it.**

3. **Follow the popup link to start configuration** or go to Settings > Admazing anytime.

4. Go to dashboard and click "Edit icons"

== Frequently Asked Questions ==

**I dont like the default icons**

Enter on the live editor and delete it. Add your own ones.

**I can't change the icons of the Dashboard Toolbar**

Go to Settings > Admazing and follow the "Live editor of the Dashboard Admazing Icons" link, or enter the URL: http://www.yourdomain.com/wp-admin/index.php?editAdmazingDash=1 in your browser

**My icon changes are lost**

Enter again on the live editor, should be there. Click on "Save changes" icon.

**I've configured my login screen with another plugin, Admazing can turn off the login customisation?**

Yes. By default Admazing comes with login customisation turned off.

**My login page shows as by default, not customised**

Go to Settings > Admazing > Login Screen tab and activate "Customize the WordPress login through Admazing" checkbox, change some color and save.

**My login page logo appears with an ugly shape background**

You should have a logo without background and save as PNG with transparency... or set the background of login screen as the background logo.
